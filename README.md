# AI-Footprint
Tracker of water and energy usage for ChatGPT

# How To Run
1. Once you have the repo, run the following commands to get localhost running
    - `npm install`
    - `npm run dev`
2. Go to `chrome://extensions`
    - Turn on developer mode
    - Click `load unpacked` and when prompted select the folder in the repo that looks like `...\AI-Footprint\.output\chrome-mv3-dev`
        - If \.output doesn't appear in the directory, try `cmd` + `shift` + `.` (on Mac)
3. Go to ChatGPT 
    - Click on the extension (puzzle-looking) icon
    - The AI-Footprint extension will appear. Click on it to make the sidepanel appear.
